Workshop Management System

About the Project

The Workshop Management System is a full-stack web application developed to optimize car service operations. It provides tools for customer and vehicle management, booking tracking, and invoicing. This project was created as part of my final-year portfolio to demonstrate proficiency in full-stack development using Laravel, JavaScript, and MySQL.

Key Features

  • Customer & Vehicle Management
  • Service Booking & Tracking
  • Role-based Access for Frontdesk & Manager
  • Invoice & Transaction History Generation
  • Responsive Interface built with Bootstrap

Tech Stack

Frontend: Bootstrap, JavaScript
Backend: Laravel, PHP, MySQL
Tools: VS Code, Git, XAMPP

My Role

I was responsible for both the frontend and backend development of the system. I designed the database schema, implemented RESTful CRUD operations, and created an intuitive user interface that adapts to both Frontdesk and Manager roles.

Learning & Outcome

Through this project, I strengthened my understanding of MVC architecture, authentication systems, and role-based access control in Laravel. I also gained hands-on experience integrating front-end design with backend APIs to create a fully functional web application.
